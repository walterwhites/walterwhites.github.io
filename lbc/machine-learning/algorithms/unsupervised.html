Algorithms

Concept

Unsupervised learning is a type of machine learning where the algorithm is trained on unlabeled data. It aims to discover patterns, structure, or relationships within the data without the guidance of labeled output.

Purpose

The purpose of unsupervised learning is to extract meaningful information from unstructured or unlabeled data. It is commonly used in tasks like clustering and dimensionality reduction, helping in data exploration and analysis.
