Data preprocessing

Concept

Standardization is a data preprocessing process in machine learning. It aims to transform data to have a mean of zero and a standard deviation of one. This facilitates data comparison and interpretation, especially when using scale-sensitive algorithms.

Purpose

The purpose of standardization is to make different features or variables directly comparable by giving them a common scale. It helps in improving the performance of machine learning models and ensures that no single feature dominates the learning process.
